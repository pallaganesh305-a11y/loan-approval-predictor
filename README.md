# 🏦 Loan Approval Prediction System

A full-stack web application that predicts loan approval status based on applicant financial details such as income, loan amount, and credit score.

Built using **Next.js** with API routes and deployed on **Vercel**.

## 📌 Features

- Full-stack Next.js application
- REST API built using Next.js API routes
- Real-time loan approval prediction
- Clean and responsive UI
- Production deployment using Vercel
- Git version control with GitHub

---

## 🧠 How It Works

The application collects:
- Annual Income
- Loan Amount
- Credit Score

Based on these inputs, the backend API processes the logic and returns a loan approval decision.
